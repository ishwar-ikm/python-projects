This GUI Quiz App is an interactive application that allows users to take a quiz by answering true/false questions retrieved from a trivia API. This app is the enchanced version of 06 quiz oop.

Features:
The app fetches true/false questions from a trivia API to provide a variety of quiz content.
The app randomizes the order of questions each time a user starts a new quiz session to enhance the quiz experience.
It has a true and false button for the user to give answer.
It gives the feedback by changing the background color into red (wrong answer) or green (correct answer).
