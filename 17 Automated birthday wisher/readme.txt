# Automated Birthday Wisher
This is a Python program that automates the process of sending birthday wishes via email. It reads birthday data from a CSV file and sends personalized emails to the recipients. The program uses the smtplib library to establish an SMTP connection and send emails.

# Features:
Automatically sends personalized birthday wishes via email<br>
Reads birthday data from a CSV file<br>
Uses customizable email templates<br>
Supports Gmail as the email service provider<br>

User can personalize the message in letter_templates and add or edit birthdays in birthday.csv file.

To completely automate this project we can upload these files to python anywhere website (https://www.pythonanywhere.com/) and schedule task to run this program every day on certain time on that day.
